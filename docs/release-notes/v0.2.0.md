# contextdiet v0.2.0

This release adds the missing CI integration layer around the original CLI.

## Added

- `contextdiet.config.json` with:
  - `threshold`
  - `ignoredRules`
  - per-rule `weight` overrides
- `init` command for creating a starter config.
- `--help` and `--version`.
- `scan --sarif` for SARIF 2.1.0 output.
- Reusable GitHub Action:

```yaml
- uses: Tehlikeli107/contextdiet@v0.2.0
  with:
    root: .
    threshold: 90
    format: text
```

- JSON schema for config files.

## Verification

- `npm test`
- `node ./bin/contextdiet.js scan --root . --strict --threshold 90`
- `node ./bin/contextdiet.js scan --root . --sarif`
- `npm pack --dry-run`

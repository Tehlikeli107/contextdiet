# contextdiet v0.1.0

Initial public MVP.

## Highlights

- Scan AI coding-agent context files for stale, noisy, contradictory, or risky context.
- Score repo context quality out of 100.
- Generate a Shields.io README badge.
- Fail CI with `--strict --threshold`.
- Apply conservative whitespace-only fixes with `fix --safe`.

## Supported Surfaces

- `AGENTS.md`
- `CLAUDE.md`
- `.cursor/rules/*.md`
- `.cursor/rules/*.mdc`
- `.github/copilot-instructions.md`
- `**/SKILL.md`
- `.mcp.json`
- `.cursor/mcp.json`

## Verification

- `npm test`
- `node ./bin/contextdiet.js scan --root . --strict --threshold 90`
- `node ./bin/contextdiet.js scan --root examples/noisy-agent-context`
- `npm pack --dry-run`

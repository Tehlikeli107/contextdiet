# contextdiet v0.3.0

This release makes contextdiet easier to share in pull requests and easier to act on after a scan.

## Highlights

- `contextdiet doctor` groups findings by impact and prints deterministic next steps.
- `scan --format markdown` creates GitHub-ready reports for issues, comments, and summaries.
- The reusable GitHub Action can now post or update one pull request comment with `comment: true`.

## Try it

```bash
npx contextdiet doctor
npx contextdiet scan --format markdown
```

## GitHub Action

```yaml
- uses: Tehlikeli107/contextdiet@v0.3.0
  with:
    root: .
    threshold: 90
    format: markdown
    comment: true
```
